This is a modern, responsive single-page website designed specifically for professional landscaping businesses. The site features beautiful design, smooth animations, mobile optimization, and lead generation focused layout.

## Quick Start

### View the Site Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Visit `http://localhost:5173` in your browser

### Build for Production

```bash
# Create production build
npm run build
```

The optimized site will be in the `dist` folder, ready to deploy.
